## Documentation détaillée (si vous avez besoin d’aide supplémentaire)

Documentation complète, incluant dépannage, précisions techniques et guides pour installations plus complexes :

https://gitlab.info.uqam.ca/fam.george/LinuxInstallGuide

Ce dépôt comprend notamment :

- Instructions détaillées étape par étape,
- Guides pour Mac, Windows, Linux, VirtualBox et UTM,
- Correctifs VirtualBox sous Windows (Visual C++ Redistributable),
- Guide Dual-Boot et considérations importantes (BitLocker, Secure Boot, pilotes récents),
- Configuration du réseau UQAM (Eduroam),
- Installation des pilotes (NVIDIA, imprimantes UQAM),
- Notes avancées pour la configuration de Debian.

Il est particulièrement utile si :

- Vous rencontrez des erreurs ou problèmes techniques,
- Vous souhaitez installer Linux en Dual-Boot ou comme système principal,
- Vous utilisez un Mac Apple Silicon.
